# AssetMind AI

AI-powered assistant designed to help organizations manage assets more efficiently by combining Large Language Models (LLM), Retrieval-Augmented Generation (RAG), and direct database integration.

## What it does
- Answers asset management questions in natural language
- Searches and understands regulations and internal documents
- Retrieves information directly from databases (MySQL)
- Assists with faster, more accurate decision-making

## Tech Stack
- LLM (Large Language Model)
- RAG (Retrieval-Augmented Generation)
- MySQL
- n8n (workflow automation)
- Laravel

## Status
This project demonstrates how AI can transform complex administrative processes into simple conversational experiences. Currently expanding into a broader platform with an AI SQL Assistant and n8n-based workflow automation modules.
